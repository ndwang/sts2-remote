  3b. Map Navigation

  13. Get state on map
  curl http://localhost:8080/state
  13. Verify: context is "map", availableNodes lists nodes with coord and type
  14. Select a map node (use a coord from availableNodes)
  curl -X POST http://localhost:8080/action -d '{"type":"select_map_node","coord":{"row":0,"col":0}}'
  14. Verify: game navigates to that room
  15. Select invalid map node
  curl -X POST http://localhost:8080/action -d '{"type":"select_map_node","coord":{"row":99,"col":99}}'
  15. Verify: returns error

  3g. Edge Cases

  30. Action outside active run
  curl -X POST http://localhost:8080/action -d '{"type":"end_turn"}'
  30. Verify: returns error, no crash
  31. Invalid JSON
  curl -X POST http://localhost:8080/action -d 'not json'
  31. Verify: returns {"error": "Invalid JSON"}
  32. Empty body
  curl -X POST http://localhost:8080/action -d ''
  32. Verify: returns error
  33. Missing type field
  curl -X POST http://localhost:8080/action -d '{"foo":"bar"}'
  33. Verify: returns error about missing type
